Q: What is Java?
A: Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

Q: What is the JVM?
A: JVM stands for Java Virtual Machine. It is a virtual machine that enables computers to run Java programs as well as programs written in other languages that are compiled to Java bytecode.

Q: What is Spring Boot?
A: Spring Boot is an open-source Java-based framework used to create stand-alone, production-grade Spring-based applications with minimal configuration.

Q: What is Dependency Injection?
A: Dependency Injection is a design pattern used in Spring to achieve Inversion of Control (IoC) between classes and their dependencies.

Q: What is REST API?
A: REST API (Representational State Transfer) is a software architectural style that defines a set of constraints to be used for creating web services.

Q: What is a microservice?
A: Microservices are an architectural style that structures an application as a collection of loosely coupled services, which implement business capabilities.

Q: What is Hibernate?
A: Hibernate is an object-relational mapping (ORM) tool for the Java programming language that provides a framework for mapping an object-oriented domain model to a relational database.

Q: What is Docker?
A: Docker is a platform for developing, shipping, and running applications inside lightweight containers.

Q: What is JWT?
A: JWT (JSON Web Token) is an open standard for securely transmitting information between parties as a JSON object.

Q: What is Feign Client?
A: Feign is a declarative web service client developed by Netflix. It makes writing web service clients easier by using annotations.
